# vue-cli3-template

## 安装工程
```
yarn install or yarn
```

### 运行开发环境并且打开浏览器
```
yarn run serve 
```

### 编译生产文件
```
yarn run build
```

### 修复eslint错误
```
yarn run lint
```
### yarn安装和卸载开发依赖
```
yarn add webpack --dev => yarn remove webpack
```
### yarn安装和卸载生产依赖
```
yarn add webpack  => yarn remove webpack
```
### yarn安装全部依赖
```
yarn or yarn install
```
### yarn初始化
```
yarn init
```
